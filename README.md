# Zee5 TV Guide Scraper

Web scraper that extracts TV program schedule data from the Zee5 TV Guide Listings

### Prerequisites
- Ruby 3.x
- Bundler

### Setup and installation
```
git clone https://github.com/ycisir/task.git
cd task
bundle install
ruby scraper.rb
cat output.json
```
